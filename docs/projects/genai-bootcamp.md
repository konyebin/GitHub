## What it is

**3-day GenAI bootcamp** repository with per-day folders and lab materials (BridgeIT / OpenAI API for self-paced use).

## When to use it

- Follow instructor-led or self-paced GenAI curriculum
- Day 2–3 labs (building, monitoring, low-code GenAI)

## Stack and entry points

| Item | Path |
|------|------|
| Overview | `GenAI-Bootcamp/README.md` |
| Day folders | `GenAI-Bootcamp/Day *` |

## Prerequisites

| Need | Detail |
|------|--------|
| LLM API | BridgeIT key ([ai-chat.cisco.com](https://ai-chat.cisco.com/bridgeit-platform/api/home)) or own OpenAI access |
| Per-lab | `pip install -r requirements.txt` in the specific day folder |

Request BridgeIT keys with educational / proof-of-concept use case; approval can take a few days.

## How to run

Open the README for the day you are on; each section links to lab folders and setup steps.
