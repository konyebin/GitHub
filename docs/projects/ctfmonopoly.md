## What it is

A **static browser mini-app** — Monopoly-style or CTF-themed game UI (`index.html`, `game.html`, `script.js`, `style.css`). No backend or package manager in the tree.

## When to use it

- Open locally in a browser for the game/demo UI
- Not integrated with Webex or other workspace MCP projects

## Stack and entry points

| Item | Path |
|------|------|
| Landing | `CTFMonopoly/index.html` |
| Game | `CTFMonopoly/game.html` |
| Logic / styles | `CTFMonopoly/script.js`, `CTFMonopoly/style.css` |

## Prerequisites

| Need | Detail |
|------|--------|
| Runtime | Modern web browser only |

## How to run

```bash
open ~/Documents/GitHub/CTFMonopoly/index.html
# or serve the folder with any static file server
```
